# MediaPad Web Config

English version: [README_en.md](README_en.md)

このディレクトリには、CH552 MediaPad の設定用 Web アプリが含まれています。  
ブラウザの WebHID を使って、デバイスのキーマップ・エンコーダ設定・マクロを読み書きできます。

## ファイル構成

- `ch552_mediaPad.html`: 画面本体
- `ch552_mediaPad.js`: WebHID 通信と UI ロジック
- `style.css`: 画面スタイル

## 主な機能

- デバイス接続（WebHID）
- キーマップ読み出し / 書き込み（2レイヤー）
- ロータリーエンコーダ（CW / CCW）割り当ての読み出し / 書き込み
- マクロ読み出し / 書き込み
- マクロ編集ダイアログ（追加、削除、クリア、容量表示）
- キーマップをデフォルトにリセット
- ブートローダー開始
- NVM ダンプ表示、macro ptrs 読み出し
- ログ表示 / クリア

## 動作要件

- WebHID 対応ブラウザ（Chrome / Edge など）
- MediaPad ファームウェアが WebHID 通信可能な状態で接続されていること
- 接続対象は以下に一致するデバイス
  - Vendor ID: `0x1209`
  - Usage Page: `0xFF82`
  - Usage: `0x51`

## 使い方

1. `ch552_mediaPad.html` をブラウザで開きます。
2. 「キーボードに接続」を押し、対象デバイスを選択します。
3. 接続後、必要に応じて次を実行します。
   - 「キーマップを読み出す」: 現在のキー / エンコーダ割り当てを取得
   - キーをクリックして割り当て変更（画面上は Pending 状態）
   - 「キーマップを書き込む」: Pending 変更をデバイスへ反映
   - 「マクロを編集する」: マクロ編集ダイアログを開いて内容を変更
   - 「マクロを書き込む」: 編集内容を EEPROM へ保存
4. 必要に応じて「デフォルトにリセット」や「ブートローダー開始」を使います。

## 仕様メモ

- SW7 は固定 Fn キー（`0xBC`）として扱われ、通常操作では変更できません。
- マクロ領域は 85 バイトです（連続したマクロ列として管理）。
- 未保存のマクロ編集がある場合、マクロ再読み出し時に確認が入ります。
- デバイス切断時は UI 状態と待機中コマンド状態をリセットします。

## 補足

- 接続時にデバイスのバージョン番号を確認しています（想定値: `0x1234`）。
- ブートローダー開始後はデバイスが切断されることがあります。
