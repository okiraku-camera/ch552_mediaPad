# Laser-cut acrylic plates and PCB assembly.

This directory contains dimensions (*.pdf) for three acrylic plates and cutout drawings (*.dxf) for manufacturing orders.

![structure](../assets/ch552_mediaPad_plate_assembly.png)


This diagram shows the assembly structure of a multi-layered setup including a PCB and acrylic plates. 

* Plates and PCB
  * Top Plate (5mm): The top layer of the assembly is a 5mm thick transparent acrylic plate.
  * PCB (1.5mm): Below the top plate is the PCB, which is a 1.5mm thick.
  * Middle Plate (5mm): Positioned beneath the PCB is another 5mm thick matte-transparent acrylic plate.
  * Bottom Plate (2mm): The bottom layer of the assembly is a 2mm thick transparent acrylic plate.

* Fastening Components
  * An 8mm M2 screw is used to secure the top of the assembly.
  * A 4mm M2 screw is used to secure the bottom of the assembly. 
  * M2 Spacer (male-female, 9mm): A 9mm long M2 spacer is placed between 2 screws, from the bottom.
